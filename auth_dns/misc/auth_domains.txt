# Authoritative DNS zone records
# Format:
#   A:       domain ip_address
#   AAAA:    domain ipv6_address
#   NXDOMAIN: domain NXDOMAIN
#   MX:      domain MX priority hostname
#   NS:      domain NS nameserver-hostname
#   CNAME:   domain CNAME canonical-name
#   TXT:     domain TXT "text data"   (quotes optional)
#   SRV:     domain SRV priority weight port target
#   SOA:     domain SOA primary-ns admin-email serial refresh retry expire minimum
#
# SOA fields:
#   serial   = YYYYMMDDNN (increment on every zone change)
#   refresh  = how often secondaries poll for updates (seconds)
#   retry    = secondary retry interval after a failed refresh
#   expire   = secondary discards zone data after this many seconds without refresh
#   minimum  = negative caching TTL (RFC 2308); also floor for all other TTLs
#
# Wildcards: use *.zone syntax for the domain (A and AAAA only)
